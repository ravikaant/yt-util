# YouTube Grid Customizer

A Chrome extension that allows you to customize the number of videos that appear in each row of YouTube's grid layout.

## Features

- Adjust the number of videos per row (between 2 and 8)
- Settings persist across browser sessions
- Updates immediately when changes are made
- Works on YouTube's home page and search results

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked" and select this extension's folder

## Usage

1. Click the extension icon in your Chrome toolbar
2. Enter the desired number of videos per row (2-8)
3. Click "Save" to apply the changes
4. The YouTube grid layout will update immediately

## Notes

- Changes apply to YouTube's desktop version only
- The extension needs to be reloaded if YouTube is already open when installing
- Minimum videos per row: 2
- Maximum videos per row: 8
